# Home Lab

## Overview

Captured and analyzed live Wi-Fi traffic in Wireshark, identifying protocols and applying display filters including DNS and IP-based filters to isolate specific traffic types.

## Tools Used

- Wireshark (packet capture and analysis on Windows, Wi-Fi interface)

## What I Practiced

- Starting a capture on a wireless interface and reading the packet list (time, protocol, length, Info).
- Recognizing common patterns: UDP/TCP, TLS 1.2 application data, HTTPS-related TCP on port 443.
- Applying Wireshark display filters and comparing total packets vs. displayed counts.
- Isolating DNS traffic and observing query vs. response lines in the list.
- Interpreting MDNS (multicast DNS) rows when filtering by IP-related criteria.
